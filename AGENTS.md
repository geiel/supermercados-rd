## Coding Instructions
- `params` and `searchParams` are **Promises**, always resolve them using `await`.
- Use `repositionInputs={false}` in the Drawer if his content have any input that needs a keyboard.
- Use the `Empty` shadcn component for empty states in the pages.
- If an drawer or dialog is jut a list for select one product use a radio button style with the text, if it is a multiple select use a checkbox with text style.
- Admin pages always will be protected and only the admin user can enter.

## Installing new libraries
- Always use `pnpm`
